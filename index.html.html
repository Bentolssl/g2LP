📈 Projeto de Análise e Visualização de Dados - Dengue no Brasil

Este projeto foi desenvolvido como parte da disciplina Linguagem de Programação — Análise e Visualização de Dados com Python. O objetivo é analisar a relação entre casos de dengue e fatores climáticos (chuva e temperatura) no período de 2015 a 2024, utilizando dados simulados.

🔗 Acesse os recursos

📊 Dashboard Interativo (Streamlit Cloud)
📓 Notebook de Análise Exploratória
🐙 Repositório no GitHub

📋 Tecnologias Utilizadas

  • Python (Pandas, NumPy, Matplotlib, Seaborn)
  • Streamlit para dashboard interativo
  • Plotly para gráficos e mapas interativos
  • GitHub Pages para página de apresentação
  • Streamlit Community Cloud para hospedagem do dashboard

✅ Funcionalidades Implementadas

  • Funcionalidades intermediárias: filtros múltiplos, KPIs dinâmicos, gráficos interativos, análise temporal.
  • Funcionalidades avançadas: correlação estatística (Pearson com lag), séries temporais (média móvel, sazonalidade), mapas interativos (choropleth por UF).
